シンプル＆見やすい！ポートフォリオHTMLテンプレート（レスポンシブ対応）

【概要】
自己紹介や経歴をわかりやすく整理できる、HTML・CSSで作られたポートフォリオ用テンプレートです。マークアップ初学者の方でも編集しやすいよう、コメント付きで設計しています。

【特徴】
HTML / CSS / JavaScript で構成された静的サイト

レスポンシブ対応（スマホ・タブレット・PCで最適表示）

編集ガイド付きコメントあり（HTMLとJavaScript内に記載）

スキルバーや経歴セクションなど、構成済み

自己紹介・職歴・スキルなどを簡単にカスタマイズ可能

GitHub Pagesなどで即公開OK！

【想定利用者】
就職活動用にポートフォリオを作りたい方

フリーランスとして自分の実績を見せたい方

HTML/CSSを学び始めたばかりの方

テンプレートをベースにカスタムして使いたい方

【ファイル内容】
/portfolio-template/
├── index.html
├── style.css
├── script.js
└── images/（カラーサンプル、画僧サンプル）
【使用方法】
任意のエディタ（VS Codeなど）で index.html を開く

コメントに従って、氏名・経歴・スキルなどを入力

ブラウザで index.html を開いて確認

必要があればGitHub Pagesなどにアップロード

【注意事項】
本テンプレートは静的HTMLサイトです。WordPress等では使用できません

商用利用・カスタマイズ可能ですが、再配布・再販売はご遠慮ください

【ライセンス】
個人・商用利用可（クレジット表記不要）／再配布・転売禁止

【サポートについて（任意）】
基本的にはサポート対象外ですが、軽微な質問には対応可能です。お気軽にご連絡ください。

【ファイルのダウンロード方法】  
✅ **購入後、Gumroadのページで「Download」ボタンをクリック**  
📥 ZIPファイル (`portfolio-template.zip`) が自動でダウンロードされます。  

💡 **解凍方法（初心者向け）**  
- **Windows** → 右クリック → **すべて展開**  
- **Mac** → ダブルクリックで解凍  

【GitHub Pagesを利用した公開手順】  
🌍 **ポートフォリオを公開する方法**  
1️⃣ **ダウンロードしたファイルをGitHubリポジトリにアップロード**  
2️⃣ `index.html` があるディレクトリで **GitHub Pages を有効化**  
3️⃣ **発行されたURLを使ってポートフォリオを公開！**  

💡 **GitHub Pagesなら、無料でポートフォリオを公開可能！**
